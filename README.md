# CSC356 Project

This project is a school assignment for CSC356. It contains a mix of PHP and JavaScript files for building a web application.

## Files

- **app.php** - Main PHP application file
- **index.php** - Entry point for the web application
- **menu.php** - Menu component
- **KittyCountdown.php** - PHP file related to Kitty Countdown functionality
- **Java.js** - JavaScript file (note: likely not actual Java)
- **Kitty.js** - JavaScript file for Kitty functionality
- **main.css** - Main stylesheet

## Setup

1. Ensure you have PHP installed
2. Serve the project from a web server
3. Open `index.php` in your browser
